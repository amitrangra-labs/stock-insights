Observability

Live SLO & metrics dashboard for Stock Insights — served by Grafana over Prometheus. Covers the two reliability planes: request serving (availability & latency) and data freshness (cache staleness).

Raw endpoints: /actuator/prometheus (scrape), /actuator/health (health), and the full dashboard directly in Grafana.

Not seeing the dashboard? Start the stack with docker compose up --build (brings up the app, Prometheus, and Grafana) — see the Observability section of the README. ← Back to dashboard
